npm基本命令

Feb 12, 2022
npm -v
  • 初始化
npm init
默认参数
npm init -y
  • 查看已安装包
当前环境中的包
npm list
系统环境中的包
npm list -g
  • 安装
npm install <Module Name>
or
npm i <Module Name>
安装到系统坏境中
npm i <Module Name> -g
安装到生产环境
npm i <Module Name> --save
or
npm i <Module Name> -S
安装到开发坏境
npm i <Module Name> --save-dev
or
npm i <Module Name> -D
  • 淘宝cnpm
npm install -g cnpm
  • 运行
npm run <Script Name>
  • 帮助
npm help
npm -l
  • 设置镜像
npm config set registry https://registry.npm.taobao.org
查看镜像
npm config get registry
