腾讯云函数入门

Jan 09, 2020

使用腾讯云控制台城建函数

之前安装onedrive云盘用的就是这种方式，装多了感觉就简单了，官方文档：https://cloud.tencent.com/document/product/583/37509

使用控制台创建函数

官方文档：https://cloud.tencent.com/document/product/583/37510 其实就是使用cmd控制台来创建，和hexo或者git操作有点类似。 这里简单说下步骤：

  1. 安装scf：pip install scf 可通过scf --version查看是否安装成功
  2. 配置账号：scf configure set 账号信息可以查看链接
  3. 初始化项目：scf init
  4. cd进入项目后本地调试运行函数scf native invoke --no-event，调试参数见链接，说一下调试环境python或nodejs均可，如果是用scf local invoke则需要docker。
  5. 编写函数和配置文件
  6. 部署scf deploy,成功后在腾讯云控制台可以查看
  7. 将 serviceId 复制到 template.yaml

VSCode插件

官方教程：https://cloud.tencent.com/document/product/583/37511 在插件市场搜索Tencent Serverless安装结果第一个即可，将之前的命令都按钮化了，更简单直观，还能在vscode里debug。
